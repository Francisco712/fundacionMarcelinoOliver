Introducción

Este proyecto, que conformará la web oficial de la Fundación Marcelino Oliver se centrará en ayudar y proteger a las víctimas de los accidentes de tráfico mediante la prestación de servicios de información y de asesoramiento de índole legal, psicológica y medica entre otros. Para ello en las diferentes pestañas de esta web se encontrará la información necesaria o en su caso, la información de contacto para poder tratarse el asunto de una forma más personal. Además también se utilizará para distribuir y gestionar una serie de noticias, ya sean noticias sobre la propia Fundación o noticias de interes para el mundo del ciclismo.

Bien, para un mejor entendimiento del funcionamiento de la página, se redacta a continuación detalladamente las funciones que hasta el momento están incluidas, en caso de que alguna funcionalidad no quede especialmente explicada siempre puede recurrir al desarrollador de la página para que le explique el caso concreto.

Permisos

Los permisos son autorizaciones para realizar ciertas acciones. Los permisos son modificables a nivel de usuario desde la pestaña "Usuarios". El unico que podrá modificar los permisos será el administrador de la web. Entre los permisos que están disponibles hasta el momento son los siguientes:

Figura 1

En la Figura 1 se pueden ver un ejemplo de permisos disponibles, los cuales se dividen en dos grupos: permisos sobre usuarios esto nos permitirá hacer o no acciones en la pestaña Usuarios y permisos sobre noticias que nos permitirán hacer acciones o no en la pestaña noticias. A continuación explicaremos detalladamente cada uno.

  • Usuarios:
      El usuario podrá acceder a la pestaña Usuarios: Si este permisos está marcado, el usuario tiene acceso a la pestaña "Usuarios" del panel izquierdo (), en caso de no estár marcado no aparecerá el icono.
      El usuario podrá crear nuevos usuarios: Si este permisos está marcado, el usuario puede crear nuevos usuarios, siendo visible el botón "Nuevo Usuario" (), en caso de no estár marcado no aparecerá.
      El usuario podrá modificar los usuarios existentes: Si este permiso está marcado, el usuario puede modificar el usuario seleccionado, estando habilitado el botón correspondiente (), en caso de no estár marcado, el botón estará deshabilitado.
      El usuario podrá eliminar los usuarios existentes: Si este permiso está marcado, el usuario puede eliminar el usuario seleccionado, habilitando el botón correspondiente (), en caso de no estár marcado, el botón no aparecerás.
  • Noticias:
      El usuario podrá acceder a la pestaña Noticias: Si este permisos está marcado, el usuario tiene acceso a la pestaña "Noticias" del panel izquierdo (), en caso de no estár marcado no aparecerá el icono.
      El usuario podrá crear nuevas noticias: Si este permisos está marcado, el usuario puede crear nuevas noticias, siendo visible el botón "Nueva Noticia" (), en caso de no estár marcado no aparecerá.
      El usuario podrá modificar las noticias existentes: Si este permiso está marcado, el usuario puede modificar la noticia seleccionada, habilitando el botón correspondiente (), en caso de no estár marcado, el botón estará deshabilitado.
      El usuario podrá eliminar las noticias existentes: Si este permiso está marcado, el usuario puede eliminar la noticia seleccionada, habilitando el botón correspondiente (), en caso de no estár marcado, el botón no aparecerás.
